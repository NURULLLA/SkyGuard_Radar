# Инструкция по деплою на Render.com

Этот проект полностью готов к бесплатному хостингу на Render. 

## Шаг 1: Загрузка на GitHub
1. Создайте новый (приватный или публичный) репозиторий на вашем GitHub.
2. Загрузите туда все файлы из этой папки.
   - *Важно: Не забудьте файлы `app.py`, `requirements.txt`, `static/`, `templates/` и `config.json`.*

## Шаг 2: Настройка на Render
1. Зарегистрируйтесь на [Render.com](https://render.com/).
2. Нажмите **New** -> **Web Service**.
3. Подключите свой GitHub аккаунт и выберите созданный репозиторий.
4. В настройках укажите:
   - **Name**: `skyguard-dashboard` (или любое другое)
   - **Runtime**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn app:app` (В Render используется именно эта команда)

## Шаг 3: Переменные окружения (Environment Variables)
Для корректной работы потоков мониторинга в Render, перейдите в раздел **Environment** в настройках вашего сервиса и добавьте:
- `PYTHONUNBUFFERED`: `1`

## Особенности бесплатного тарифа
- Сайт будет засыпать через 15 минут бездействия.
- Для первого открытия после сна может потребоваться около 30-50 секунд.
- База данных (последние позиции самолетов) будет очищаться при каждом перезапуске сервера (это происходит примерно раз в день), но так как вы сказали, что история не важна, это не создаст проблем.

Поздравляю! Теперь ваш дашборд будет доступен по ссылке вида `https://skyguard-dashboard.onrender.com`.
